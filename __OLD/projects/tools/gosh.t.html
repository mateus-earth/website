Gosh
Description:

Gosh is shell bookmarker tool. Shortly it let's you to sets a given directory path as a simple name that can be used to cd more easily.

Among the increased usability we have a fuzzy match for names, so it's not required to type exactly the name of the bookmarker and the ability of use it in shell subexpressions.
